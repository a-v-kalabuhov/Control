# UsrConnector — нижний слой коннектора ТПА

Read-only опрос IoT-шлюза USR (Modbus TCP) на термопластавтомате BORCHE/KEBA: съём сигналов
цикла литья, вычисление состояния машины (`MachineState`), выдача наверх. Верхний слой
(шаблон устройства, упаковка сообщений, транспорт MQTT/REST) реализуется отдельно в
`UsrConnector.Host` и НЕ входит в зону этого кода.

## Жёсткие инварианты — не нарушать

1. **Read-only.** Только читающие функциональные коды Modbus (0x01–0x04). Никакой записи
   в устройство, ни при каких доработках.
2. **Граница Core / Host.** `UsrConnector.Core` не знает о транспорте (MQTT, REST, HTTP) и
   не получает таких зависимостей. Всё транспортное — только в Host.
3. **Граница stateless / stateful.** Конфиг и декодер выполняют только преобразование
   «регистр → мгновенное значение» (scale/offset или valueMap). Любая логика с памятью
   между опросами (циклы, минимумы, таймауты, статистика) — только в
   `MachineStateMachine` и процессорах.
4. **Семантические роли.** Прикладная логика обращается к сигналам по ролям
   (`SignalRole`), никогда — по адресам регистров или именам тегов.
5. **Автомат = код + тесты + документ.** Семантика `MachineMode`, `CycleCounter`,
   `CycleCompletion` меняется только синхронно в `MachineStateMachine.cs`,
   `MachineStateMachineTests.cs` и `STATE_MACHINE.md`. Тесты — исполняемая спецификация;
   «починить тест под код» без обновления документа — ошибка.
6. **Коннектор снимает, не интерпретирует.** Данные отдаются как есть (например, подушка
   публикуется и для нештатных циклов); решения об отбраковке данных — за внешними
   системами.

## Карта документов

| Документ | Что там |

| `STATE_MACHINE.md` | спецификация автомата состояния: режимы, переходы, таймауты, подушка |
| `ARCHITECTURE.md` | архитектура слоёв, конфиг, точки расширения |
| `CYCLOGRAM_SINGLE_NODE.md` | предметная область: фазы цикла литья, сигналы, точки съёма (одноузловая машина) |

## Контекст предметной области (минимум)

Цикл литья: смыкание формы → впрыск → выдержка → набор дозы → охлаждение → размыкание →
выталкивание. **Якорь цикла — впрыск** (сигнал `Injection`): его фронт начинает цикл и
инкрементирует счётчик износа формы. Штатное завершение — `EjectorFwdReached` (E1).
**Подушка** (cushion) — минимум положения шнека за цикл, ключевой показатель стабильности
процесса. Детали — в циклограммах.
